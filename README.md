# yenbekbay.me

My [personal website](https://yenbekbay.me). Powered by [hexo](https://hexo.io/) with [my custom theme](https://github.com/yenbekbay/clean-blog).

## Setup

```bash
git clone https://github.com/yenbekbay/yenbekbay.me.git
cd yenbekbay.me
yarn
yarn dev
```
