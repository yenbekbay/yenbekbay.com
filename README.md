My Blog
=======

My [blog](https://blog.yenbekbay.me). Powered by [hexo](https://hexo.io/) with [my custom theme](https://github.com/yenbekbay/clean-blog).

Setup
-----

```bash
git clone https://github.com/yenbekbay/blog.yenbekbay.me.git
cd blog.yenbekbay.me
npm install
hexo server
```
